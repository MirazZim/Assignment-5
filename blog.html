1. What is the Document Object Model (DOM)?

The DOM is a programming interface for HTML and XML documents. It represents the page as a tree structure where each node is an object.

2. How do you select an element from the DOM?

You can use methods like getElementById, querySelector, or getElementsByClassName to select elements from the DOM.

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events from child elements. It is useful for handling dynamic elements and improving performance.
